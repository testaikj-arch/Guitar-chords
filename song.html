Open Google Sheets. Add columns: ID Title Artist Key Capo Content
2
Go to Extensions → Apps Script, paste this function and save:

function doGet(){
 var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
 var r=s.getDataRange().getValues();
 var h=r[0];
 var d=r.slice(1).map(row=>Object.fromEntries(h.map((k,i)=>[k,row[i]])));
 return ContentService.createTextOutput(JSON.stringify(d)).setMimeType(ContentService.MimeType.JSON);
}
3
Click Deploy → New deployment. Type: Web app. Execute as: Me. Access: Anyone. Deploy and copy the URL.
4
Paste the URL above and click Save & Load.
