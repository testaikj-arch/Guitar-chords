⚙️ Settings

Paste your Web App URL here. This connects Chord Stage to your Google Sheet song library. The URL is saved in your browser — no code editing needed.

1
Open Google Sheets with your songs. Columns must be: ID Title Artist Key Capo Content
2
In Sheets go to Extensions → Apps Script. Paste this code and save:

function doGet(){
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var rows = sheet.getDataRange().getValues();
  var headers = rows[0];
  var data = rows.slice(1).map(r => Object.fromEntries(headers.map((h,i)=>[h,r[i]])));
  return ContentService.createTextOutput(JSON.stringify(data)).setMimeType(ContentService.MimeType.JSON);
}
3
Click Deploy → New deployment. Type: Web app. Set Execute as: Me and Who has access: Anyone. Click Deploy and copy the URL.
4
Paste that URL in the field above and click Save & Load.
